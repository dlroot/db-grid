### P1-1 Cell Data Types 自动推断
- **新增文件：** `cell-data-type.service.ts`
- `inferColumnTypes()` — 采样 rowData 推断每列类型
- 支持类型：text/number/date/boolean/object
- 内置预设：`agTextColumn/agNumberColumn/agDateColumn/agBooleanColumn`
- `applyColumnTypes()` — 处理 GridOptions.columnTypes（AG Grid 兼容）
- `applyDefaultColDef()` — 合并 defaultColDef
- `applyInferredType()` — 将推断结果写入 columnDefs
- 内置格式化器：`numberFormatter/dateFormatter/numberComparator/dateComparator`
- 集成到 `DbGridComponent.ngOnInit` 和 `setRowData`
- `getCellDataTypeService()` 暴露到 gridApi

### P1-2 Column Group Headers
- **新增逻辑：** `header-renderer.ts` 的 `render()` 方法支持分组表头
- `buildGroupCells()` — 递归构建多行分组表头
- `createGroupHeader()` — 分组单元格渲染
- `countLeafColumns()` / `calculateGroupWidth()` — 辅助计算
- **ColumnService 新增：** `getColumnTree()/hasColumnGroups()/getGroupDepth()`

### P1-3 Keyboard Navigation
- **新增文件：** `keyboard-navigation.service.ts`
- 完整键盘支持：Arrow/Tab/Enter/F2/Escape/Page/Home/End
- `KeyboardCellPosition` 接口（避免与 range-selection.service 的 CellPosition 冲突）
- `onFocusChange/onCellEditStart/onCellEditStop/onEnterKey/onEscapeKey` 事件流
- 编辑模式：Enter/F2 开始，Enter/Tab 提交，Escape 取消，方向键先提交再移动
- `highlightFocusedCell()` — CSS class 方式高亮焦点单元格
- `DbGridComponent` 集成：`onKeyDown` 事件处理、初始化、订阅
- `getKeyboardNavigationService()` 暴露到 gridApi
