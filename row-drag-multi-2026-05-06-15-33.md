# 行拖拽 Multi-Row 支持 - 任务总结

## 目标
为 DB Grid 添加 `rowDragMultiRow` input 属性以支持多行拖拽功能。

## 问题
之前的调查发现 row-drag.service.ts 存在但 db-grid.component.ts 中缺少对应的 input 属性和初始化代码。

## 修改内容

### db-grid.component.ts

1. **添加 @Input() rowDragMultiRow**：
```typescript
@Input() rowDragMultiRow: boolean = false;
```

2. **导入 RowDragService**：
```typescript
import { ..., RowDragService } from '...';
```

3. **添加私有属性**：
```typescript
private rowDragService: RowDragService;
```

4. **constructor 中初始化**：
```typescript
this.rowDragService = new RowDragService();
```

5. **ngOnInit 中初始化拖拽服务**：
```typescript
this.rowDragService.initialize({
  rowDragEnabled: this.rowDragEnabled,
  rowDragMultiRow: this.rowDragMultiRow,
});
```

6. **ngOnChanges 中处理变更**：
```typescript
if (changes['rowDragEnabled'] || changes['colDragEnabled'] || changes['rowDragMultiRow']) {
  this.rowDragService.initialize({
    rowDragEnabled: this.rowDragEnabled,
    rowDragMultiRow: this.rowDragMultiRow,
  });
}
```

7. **gridApi 添加 getRowDragService**：
```typescript
getRowDragService: () => this.rowDragService,
```
